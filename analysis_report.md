# E-commerce Sales Analysis

## Project Overview
This project analyzes sales data from an e-commerce platform to understand trends, top-selling products, and regional performance. The analysis includes data cleaning, exploration, visualization, and key insights.

## Dataset
The dataset used: `data/sales_data.csv`

Columns in the dataset:
- Date: Date of sale
- Product: Product name
- Quantity: Number of items sold
- Price: Price per item
- Customer_ID: Unique customer identifier
- Region: Sales region
- Total_Sales: Total sale amount (Quantity × Price)

## Visualizations
- Bar Chart: Total Sales by Product
- Line Chart: Monthly Total Sales Trend
- Pie Chart: Total Sales Distribution by Region

## Insights
- Top-selling product
- Region contributing most to sales
- Month with highest sales
- Total sales and total quantity sold

## Folder Structure
- `data/` → Contains the dataset
- `visualizations/` → Saves all charts as images
- `report/` → Contains the final report
- `main.py` → Python script with full analysis
- `requirements.txt` → Required Python packages
