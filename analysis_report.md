This is a beginner-friendly **Data Analysis mini-project** using Python and the **pandas** library.  
The project analyzes a simple sales dataset to extract meaningful insights such as total revenue, average sales, and the best-selling product.

---

## Project Description

The program loads a sales dataset from a CSV file and performs basic data analysis tasks such as:
- Exploring the dataset
- Handling missing values
- Calculating important sales metrics

Finally, it generates a clean, formatted sales report that summarizes key insights from the data.

This project helps in understanding **pandas basics**, **data cleaning**, **grouping operations**, and **basic reporting** in Python.

---

## Features

- Uses **pandas** to load and analyze CSV data  
- Displays dataset shape, columns, and data types  
- Checks and handles missing values  
- Calculates multiple sales metrics  
- Identifies the best-selling product  
- Generates a clean and readable sales report
